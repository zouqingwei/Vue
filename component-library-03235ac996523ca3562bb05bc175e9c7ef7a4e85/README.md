# component-library

## Project introduction
```
对于开发者来讲，组件的好处无需多言，这个`组件库`主要是本人在开发过程中对于常用功能或者是比较特殊的需求进行统一的规范整理，
在这里，每个组件都给出了相应的示例、对应的数据结构，使子组件、父组件之间的交互传参一目了然，便于广大开发者集成使用，后续
还会加入更多新的组件并不断进行优化，如果对您有所帮助，希望奉献一颗小星星！
```

## Components
```
- 顶部导航栏
- 搜索
- 时序图
- 预览单张图片
- 预览多张图片
- 预览视频（真实地址）
- 预览音频（blob类型，即不暴露真实地址）
- 文件上传进度
- 数字和字母组合的验证
- 图片形状滑动验证
```
